## Usage
1. Copy this repository.
```
git clone https://github.com/igorpichnenko/ant.git
cd ant
```
2. Install dependencies
```
yarn install
```

3. Run app
``` bash
# Dev
yarn dev

# Production
yarn build
yarn start
```
